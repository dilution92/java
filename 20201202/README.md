 ## index(subject) | find()
 - index() : 지정된 요소를 검색하여 위치값을 반환함.  0 base이며 찾지 못하면 -1을 반환.
 - find() : 저정된 요소를 검색하여 배열 타 입으로 반환
    - subject : 찾고자 하는 요소. element
  

  ## .val()
    `<input value='...'>`의 value 값을 가져온다.
