# 2. 도메인

1. 주소창에 도메인(https://google.com) 입력
2. 로컬(자신의 컴퓨터) 머신에 등록된 DNS에게 해당 도메인 질의(query)
   1. 해당 DNS에 캐싱되어 있으면 바로 응답
   2. 없으면 상위 네임 서버에 물어봄
3. 응답받은 IP 주소로 통신 (TCP/IP)

# *리눅스
1. 프로세스 목록

ps -aux

2. 프로세스 목록에서 원하는 프로그램 찾기

예) 자바 찾기
ps -aux| grep java

3. 프로세스 죽이기

kill -9<pid>

4. 백그라운드로 프로세스 실행

nohub <실행할 명령 라인>&
nohub java -jar app.jar &
